# Student Management System

## Project Title
Student Management System

## Description
Student Management System is a backend-based web application developed using Java Servlets, JDBC, and MySQL. The application allows users to manage Student records through CRUD (Create, Read, Update, Delete) operations.

The backend is connected to a MySQL database for permanent data storage. Postman is used as the client to send HTTP requests and test the REST-style API endpoints.

### Key Features
- Add a new Student
- View all Students
- Find an Student by ID
- Update Student details
- Delete an Student
- Store Student data permanently in MySQL
- Perform CRUD operations using Java Servlets and JDBC

## Technologies Used
- Java
- Java Servlets
- JDBC
- MySQL
- Apache Tomcat 9
- Eclipse
- Postman
